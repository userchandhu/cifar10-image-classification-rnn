# CIFAR-10 Image Classification using RNN/LSTM

## 📌 Overview
This project explores using RNN/LSTM for image classification on CIFAR-10 dataset.

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras

## ⚙️ Approach
- Implemented RNN with LSTM layers
- Resolved architecture issues using return_sequences
- Trained model with multiple epochs

## 📊 Results
- Achieved ~59% accuracy
- Observed performance limitations

## 🧠 Key Learning
RNN is not suitable for image data; CNN is more effective.
